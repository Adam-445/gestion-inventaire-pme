# Inventory Management System

An inventory management application designed for small to medium businesses.  
This project was built as part of the **Advanced Python Module**


## Overview

The application allows users to manage **products**, **categories**, and **stock movements (entries/exits)** through agraphical interface.  
It also includes a **dashboard** that provides key statistics such as total stock value, number of products, and alerts for low inventory levels.



### Running the application
```bash
# Clone the repo
git clone https://github.com/Adam-445/inventory-management-tkinter.git
cd inventory-management-tkinter

# Run the application
python main.py
````

The database (`inventaire.db`) will be automatically created on first launch.
